Tags em HTML

Negrito e Itálico

Usamos as tag "strong" para colocar em negrito

Usamos as tag "em" para colocar em Itálico

Marcar algo com marca-texto

Usamos as tag "mark" para marcar com marca-texto algo

Texto pequeno e grande

Para letras "grandes" se usa tag big, porém não está em uso mais

Para letras "pequenas" se usa a tag small, e sim, ainda está em uso

Texto riscado/deletado

Usamos as tag "del" para falar que o texto está excluido/não é pra ser considerado

Texto inserido

Usamos as tag "ins" para sublinhar uma parte de um texto; ou tambem se usa a tag "u" para sublinhar também porém não é semântica

Texto subrescrito

Usamos a tag "sup" para sobrescrever algo, coomo X2+15

Texto subscrito

Usamos a tag "sub" para subscrever algo, como H2O
