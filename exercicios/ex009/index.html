Trabalhando com listas

Listas Ordenadas

Usamos a Tag "ol" para dizer que será uma lista ordenada, para cada item citado na lista usamos a tag "li". Podemos dizer qual o type (tipo) usando I, i, A, a e o 1.

  1. Acordar
  2. Tomar café da manhã
  3. Faculdade
  4. Almoçar
  5. Academia
  6. Estudar
  7. Dormir

Listas Desordenadas

Usamos a tag "ul" para dizer que aquela lista será desordenada, para cada item citado na lista usamos a tag "li".Também podemos dizer qual o type (tipo) usando circle, square e disc

Listas de Definição

As listas de definição são para dar significado a determinado termo. Usamos a tag "dl" para dizer que é uma lista de definição, para o termo referido usamos a tag "dt" e para o significado/definição do termo usamos a tag "dd".

HTML5:
Linguagem de Marcação de hipertexto criação do conteúdo do site.
CSS3:
Linguagem de Marcação criação do designer do site.
Javascript:
Linguagem de Programação usada para as interação do site.
